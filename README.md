This website is a school project I did about societal problems, and my group decided we would talk about mental health, and more specifically about the mental health of teenagers and children. Unlike everyone else, instead of doing a simple slide presentation and research, we decided to create a website that will actually stand up, about a fictional company called "Mente Viva" as a way to simulate what would happen in real life. The website has a very interactive interface, with pages for children, pages for teenagers, and a page for parents and guardians. It was programmed using CSS and HTML. The use of AI was solely and exclusively for the translation of certain words and programming of parts that I had difficulty with, nothing more.

We have reached the end of the development phase for Mente Viva—originally a school project—and the next step is to help it grow on a larger scale and finally achieve the project's true objective.
I am counting on the help of all my friends to turn Mente Viva into a reality.
